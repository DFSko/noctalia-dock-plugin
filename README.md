# noctalia-dock-plugin

A side dock plugin for Noctalia, inspired by Dash to Dock.

## Features

- Exclusive dock area on the left side
- Vertical layout with icons starting from top
- Fixed on the left side
- Launches apps via `gtk-launch`
- Pinned apps are synced from launcher pins (`Settings.data.appLauncher.pinnedApps`)

## Notes

- Manage pinned items in the launcher pin/unpin actions.
- Dock list is read-only in plugin settings.
